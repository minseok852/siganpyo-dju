# 대진대학교 교양필수 강의시간표 이미지 → JSON 변환 프롬프트

당신은 대학교 강의시간표 이미지를 JSON으로 변환하는 작업을 합니다.

## ⚠️ 필수 조건
**이미지에 있는 모든 과목을 빠짐없이 전부 변환해야 합니다. 일부만 하고 멈추지 마세요. 끝까지 완료하세요.**

## 입력
- 교양필수 강의시간표 이미지 (표 형식)

## 출력 형식

```json
{
  "category": "교양필수",
  "updated_at": "2026-01-28",
  "semester": "2026학년도 1학기",
  "courses": [
    {
      "course_code": "교과번호 6자리",
      "course_name": "교과목명",
      "credits": 학점(숫자),
      "hours": 강의시간(숫자),
      "lab_hours": 실습시간(숫자, 없으면 0),
      "sections": [
        {
          "section": "분반 2자리 (01, 02 등)",
          "professor": "담당교수",
          "schedule": {
            "raw": "강의시간 원본 (예: 월10:00-11:30)",
            "days": ["월"],
            "times": [
              {"day": "월", "start": "10:00", "end": "11:30"}
            ]
          },
          "room": {
            "raw": "강의실 전체 (예: 인119-3 - 교양A/V강의실)",
            "building": "건물약칭 (예: 인)",
            "building_full": "건물전체명 (예: 인문학관)",
            "room_number": "호실 (예: 119-3)",
            "room_type": "강의실 유형 (예: 교양A/V강의실)"
          },
          "capacity": 정원(숫자),
          "notes": "비고 내용 (없으면 빈 문자열)"
        }
      ]
    }
  ]
}
```

## 중요 규칙

### 1. 모든 과목/분반 필수
이미지의 모든 행을 빠짐없이 변환하세요. 중간에 멈추거나 "나머지는 같은 방식으로..." 하지 마세요.

### 2. 같은 과목 다른 분반
같은 course_code면 sections 배열에 추가:
```json
"sections": [
  {"section": "01", "professor": "김철수", ...},
  {"section": "02", "professor": "김영희", ...},
  {"section": "03", ...}
]
```

### 3. 교수명이 여러 분반에 걸쳐 있는 경우
이미지에서 교수명이 한 번만 표시되고 여러 분반에 적용되는 경우가 있습니다.
예: 19, 20번 분반이 같은 교수 "김연희"
→ 각 분반에 동일한 교수명을 넣으세요.

### 4. 강의실이 여러 분반에 걸쳐 있는 경우
교수명과 마찬가지로 강의실도 여러 분반에 공통으로 적용될 수 있습니다.
→ 각 분반에 동일한 강의실 정보를 넣으세요.

### 5. schedule 구조
- raw: 원본 텍스트 그대로
- days: 요일만 배열로
- times: 각 시간을 객체로 분리

### 6. room 구조
- raw: 원본 텍스트 전체
- building: 건물 약칭
- building_full: 건물 전체명 (아래 목록 참고)
- room_number: 호실 번호만
- room_type: 강의실 유형

### 7. capacity (정원)
표의 "정원" 열에 있는 숫자를 그대로 입력

### 8. lab_hours
표에서 "실" 열에 있는 숫자. 없으면 0

### 9. 비고(notes)
- "OO학과 분반", "OO전공 분반" 등 수강 대상 정보
- 줄바꿈 없이 한 줄로

### 10. 홀수/짝수 주의 ⚠️
비고에서 **"홀수", "짝수"**를 절대 **"졸수", "작수"**로 잘못 읽지 마세요.

### 11. 교수명 주의사항 ⚠️
- 한글 이름: 비슷한 글자 주의 (희/의/회, 현/연/원, 수/주/우 등)
- 외국인 이름: 대문자 유지, 띄어쓰기 원본 그대로
- 불확실하면 이미지 원본을 최대한 정확히 작성

### 12. 알파벳/한글 구별 주의 ⚠️
- "AI시대의컴퓨팅사고" → AI는 영문 알파벳
- "사고와표현" → 사는 한글
- 문맥상 영어 약자인지 한글인지 판단할 것

## ⚠️ 최종 확인사항
1. 이미지의 **모든 과목과 모든 분반**을 변환했나요?
2. 같은 과목의 **다른 분반**은 sections 배열에 넣었나요?
3. **교수명/강의실**이 여러 분반에 걸쳐있는 경우 각각 넣었나요?
4. **정원(capacity)** 숫자를 정확히 입력했나요?
5. **홀수/짝수**를 정확히 읽었나요?
6. **건물 약칭**을 올바른 건물명으로 매칭했나요?

모든 조건을 만족하면 JSON을 출력하세요.
